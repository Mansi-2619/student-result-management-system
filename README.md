# Student Result Management System

A Python-based application to manage student results efficiently. This system allows you to add student details, record marks, calculate grades, and view all results. All data is stored persistently in a text file for easy retrieval.

---

## Features

- Add student details and marks for 5 subjects  
- Automatically calculate:
  - Total marks  
  - Percentage  
  - Grade (A, B, C, D, Fail)  
- View all student results in a structured format  
- Store results in a text file (`results.txt`) for persistence
